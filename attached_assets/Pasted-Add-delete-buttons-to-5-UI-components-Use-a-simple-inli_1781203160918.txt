Add delete buttons to 5 UI components. Use a simple inline trash icon button (✕ or 🗑) that confirms with window.confirm before calling DELETE, then removes the item from local state optimistically.

--- 1. apps/chg-rehab/components/workspace/TodoTab.tsx ---

In the open tasks row render, after the rowRight div, add:
<button
  type="button"
  onClick={async () => {
    if (!window.confirm("Delete this task?")) return;
    setTasks(prev => prev.filter(x => x.id !== t.id));
    await fetch(`/api/workspace/tasks/${t.id}`, { method: "DELETE" });
  }}
  style={{ background: "none", border: "none", cursor: "pointer", color: "var(--quill)", fontSize: 14, padding: "0 4px", flexShrink: 0 }}
  title="Delete task"
>✕</button>

Do the same in the done tasks row.

Also add delete for PM tasks rows (the pmTasks section): call DELETE /api/pm/tasks/[id] and filter from setPmTasks.
